Prerequisite

To run this page you need to disable your browser's web security.


You can do this by closing your browser completely:

For Windows machine\'s chrome browser:

"C:\Program Files (x86)\Google\Chrome\Application>chrome --disable-web-security"

For Mac machine\'s chrome browser:

"$ open -a Google\ Chrome --args --disable-web-security"

For Ubuntu machine\'s chromium browser:

"$ chromium-browser --disable-web-security"'

Specific Leads List View

In following way you can make a REST call To get data for a specific list view by it's id. For this you can make a GET request with a parameter 'current_listview' as view id. You can also do it by a POST call also with an extra parameter '_method' with value 'index'.

Specific lead listview REST call snapshot

Use the GET request with a parameter 'current_listview'. You can do it by a POST call also with an extra parameter '_method' with value 'index'.
