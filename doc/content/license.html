License Management

The License Management page allows administrators to manage their MaaxFrame license properties. You can use this option to manage the MaaxFrame license for your organization and view information such as the download key, expiration date, number of users, etc. System administrators can also edit and re-validate the license via the License Management page. Please note that your commercial edition of MaaxFrame is purchased with a set number of licenses that allow users to access the application.



to see the licensing validity for a module you can open the form view of a particular module and you can see the start date and end date and days to expire


Creating a License

Navigate to the Settings -> Configuration -> User License.



Click on create new it will open up a license form. now select module name from the module drop down and select user for whom you want to create a license. now click submit. Again open the record in form view mode you will see the link for downloading license. click on download link it will download a .ml file


to deploy the installed license just go to the module for which you have created license. open module in form view. Click on deploy and install license and you are done you will find the module information updated.

Download Key

The download key is a 32 digit alphanumeric key that is unique to your MaaxFrame account. The number of user licenses your organization purchased, as well as the subscription expiration date is tied to your download key. Upon purchasing MaaxFrame, you will receive a Welcome email containing your download key information which will need to be entered during installation of the MaaxFrame application. Once the download key is entered in the system, the corresponding information such as the expiration date, number of users, number of offline client licenses, and number of concurrent self-service portal users will automatically populate and be displayed in Admin > License Management.
